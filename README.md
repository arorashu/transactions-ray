# Transactions in Ray

## Report
You can view the paper presenting the report [here](https://github.com/arorashu/transactions-ray/blob/master/Shubham_Arora-Transactions_on_Ray.pdf)


## Instructions to run

1. Ensure you have python 3.6+ installed

2. Create python virtual env and install dependencies
```
    $ python3 -m venv .ray-env
    $ source .ray-env/bin/activate
    $ pip install -r requirements.txt
```

3. Run Trip Service
```
    $ python app_service.py
```
