# Transactions in Ray

## Instructions to run

1. Ensure you have python 3.6+ installed

2. Create python virtual env and install dependencies
```
    $ python3 -m venv .ray-env
    $ source .ray-env/bin/activate
    $ pip install -r requirements.txt
```

3. Run Trip Service
```
    $ python trip_service.py
```
